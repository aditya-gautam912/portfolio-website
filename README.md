# Aditya Gautam - AI & Data Science Portfolio 🚀

A modern, responsive, and professional portfolio website built with Vanilla HTML, CSS, and JavaScript. This site is designed to showcase my projects, skills, and experience as an AI & Data Science Developer.

![Portfolio Preview](assets/images/profile.png)

## 🌐 Live Demo
[https://aditya-gautam912.github.io/portfolio-website/](https://aditya-gautam912.github.io/portfolio-website/)

## ✨ Features
- **Dark/Light Mode Toggle**: User preference saved in local storage.
- **Scroll Reveal Animations**: Smooth entry effects using Intersection Observer API.
- **Animated Skill Bars**: Visual proficiency levels.
- **Responsive Design**: Mobile-first approach for all devices.
- **Project Showcase**: Direct links to my top GitHub repositories.
- **Resume Integration**: Easy-to-download PDF resume.

## 🛠️ Built With
- **HTML5**: Semantic structure.
- **CSS3**: Custom variables, Flexbox, and Grid.
- **JavaScript (ES6)**: Clean, library-free interactivity.
- **Font Awesome**: High-quality vector icons.
- **Google Fonts**: Inter & Poppins typography.

## 📂 Project Structure
```text
portfolio/
├── assets/
│   ├── css/      # Custom styles and theme variables
│   ├── js/       # Interactive logic and animations
│   ├── images/   # Profile and project screenshots
│   └── docs/     # Professional resume (PDF)
└── index.html    # Core structure and content
```

## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/aditya-gautam912/portfolio-website.git
   ```
2. Open `index.html` in your browser.
